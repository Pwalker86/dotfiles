Clone to ~/.config/

If on a windows WSL installation, the paths in .zshrc aren't correct. Not hard to correct, but be aware.

```bash
symlink
 - ln -s ~/.config/dotfiles/init.vim ~/.config/nvim/init.vim
 - ln -s ~/.config/dotfiles/.zshrc ~/.zshrc

```
